Site Amaci

Bu web sitesi, Mesleki ve Teknik Anadolu Lisesi Bilisim Teknolojileri Alani 11. sinif Web Tabanli Uygulama Gelistirme dersi kapsaminda hazirlanmistir.

Sitemizin amaci:

  • Web tasarim ilkelerini ogrenmek
  • HTML5, CSS3 ve Bootstrap konularinda ornekler sunmak
  • Ogrencilere kaynak olusturmak
  • Web programlama becerilerini gelistirmek

Misyonumuz

Ogrencilere modern web teknolojilerini ogretmek ve uygulamali projelerle pekistirmek. Web tasariminda en iyi uygulamalari takip ederek kaliteli icerikler uretmek.

Vizyonumuz

Web programlama alaninda kendini gelistirmek isteyen ogrencilere rehber olmak. Gelecegin web gelistiricilerini yetistirmeye katkida bulunmak.
